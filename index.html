What is CodeMan ?

CodeMan is a Discord bot to manage Slippi connect codes for SSBM.

Features

  • Find the discord nametag of someone that is in the database.
  • Show your code in a very quick way.
  • The code database is cross-server.
  • Slippi API integration.

How does it work ?

It is based on user participation which means the more users are connected with
CodeMan the more powerful CodeMan will be. To connect see the &connect
command in the next section.

Command list

connect

Connect through Slippi by using your connect code.

&connect [code]

code

Shows your code or the one of someone else.

&code (optional discord tag)

whois

Finds a discord username from a code or a slippi username.

&whois [slippi username or code]

disconnect

Wipes all your data from CodeMan's database.

&disconnect

help

Shows a command list like this one.

&help
